# OnlineEducation项目说明

## 项目安装运行依赖
```
npm install
```

## 编译和热更新（用于开发环境）
```
npm run serve
```

## 编译并压缩（用于生产环境）
```
npm run build
```

## 检查并修复源文件代码规范性
```
npm run lint
```

## 参考官方文档
查看 [ @vue/cli 的使用文档 ](https://cli.vuejs.org/config/).
